About

PinItTo.me is an infinite virtual corkboard which can be put to use in many different ways!

Best of all PinItTo.me is open source so you are welcome to make improvements to the system or run your own instance.

We’re currently still in alpha but hope to improve the system over the coming weeks and get a stable product out there. In the meantime please feel free to use and let us know about any problems.
